# Apex Code Snippet Generator

Instantly generate common code snippets in various languages using natural language requests. Designed for developers, students, and freelancers seeking quick, AI-powered boilerplate code. Input your need, get your code.

## Features

*   **Natural Language Input:** Describe the code you need.
*   **AI-Powered Generation:** Get relevant code snippets quickly.
*   **Web UI:** Simple and intuitive interface.

## How to Use (Local Development)

1.  Save `index.html` to your computer.
2.  Open `index.html` in your web browser.
3.  Type your code request into the text area and click "Generate Snippet".

**Note:** This is a frontend-only demonstration. A full implementation requires a backend API for AI code generation and payment processing.
